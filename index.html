Amptools.Net is an online hub of opensource software development tools, tutorials, and other random bits of geekdom.

Projects

  • Midori-PHP - a generative php framework built on top of zend in order to get it to work like rails.
  • Midori - a generative mono/.net framework built on top of asp.net mvc in order to get it to work like rails. (still in the design/planning phase).
  • TAB - (Tools Automation Buildkit) for mono/.net (still in the design/planning phase)
